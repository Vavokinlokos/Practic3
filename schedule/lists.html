• Запросы:
    1. Составить расписание для заданной группы на каждый день недели и подсчитать общее количество пар за неделю для этой группы;
    2. Составить расписание для каждого преподавателя с заданной степенью на каждый день недели, среди этих преподавателей найти самого загруженного;
    3. Составить расписание для каждой лекционной аудитории на каждый день недели и найти лекционные аудитории, в которых вообще не проводятся лекции по заданному предмету;
    4. Подобрать замены для заданного преподавателя на заданный день недели и заданную пару, т. е. найти свободных в это время преподавателей, умеющих вести нужный предмет;
    5. Подобрать, в какие аудитории (пустые аудитории того же типа) могут быть перенесены занятия из заданной аудитории в указанный день недели на указанной паре.
  • Транзакции:
    1. Назначить новое занятие;
    2. Изменить тип аудитории или исключить ее из расписания.
